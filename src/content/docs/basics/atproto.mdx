AT Protocol - це протокол для створення децентралізованих соціальних мереж.

В першу чергу був створенний для [Bluesky](./bluesky).

Важливими особливостями є можливість алгоритмічної видачі та зміна провайдера для ваших даних без
втрати хендла (ніку).

Більше інформації в [офіційній документації](https://atproto.com/guides/overview).
