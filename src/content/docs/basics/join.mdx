Щоб зареєструватись у Bluesky, створіть обліковий запис на [bsky.app](https://bsky.app). _(Інвайти більше не потрібні!)_

Ви можете завантажити додаток Bluesky для [iOS](https://apps.apple.com/us/app/bluesky-social/id6444370199) або [Google Play](https://play.google.com/store/apps/details?id=xyz.blueskyweb.app), або використовувати [Bluesky через браузер](https://bsky.app/).
